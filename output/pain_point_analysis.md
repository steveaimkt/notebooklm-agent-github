## 🟢 확정 Pain Point (3소스 일치)

### 1. 킥백 사고 + 손목 피로도 — "다칠까 무서워서 첫 시동이 떨림"

**① 발생 사용 상황**
톱날 끝(팁)이 단단한 옹이·금속 못·다른 나무에 닿는 순간 톱이 작업자 방향으로 강하게 튀어 오르는 **킥백 현상**. 또한 한 손으로 장시간 들고 작업 시 손목 부담 누적.

**② 가장 심각하게 느끼는 고객 유형**
- **김지영 페르소나(38세 DIY 정원, 비중 33%)** — 톱 사용 처음, 입문자라 안전 민감도 최고
- **정한수 페르소나(67세 시니어 목공, 비중 17%)** — 시니어 반사신경 부담
- **한국소비자원 통계 49.4%** — 기계톱 안전사고 356건 중 팔·손 부상이 절반

**③ 경쟁사(DEWALT) 대응 — ✅ 적극 다룸 (벤치마크 포인트)**
- 상세페이지 콜아웃 4개 중 2개가 안전 관련:
  - **철재 팁 가드(Tip Guard)**: "킥백 위험 대비, 진동 감소"
  - **상부 보조 핸들 + 오버몰드 핸들**: "양손 그립으로 안정적, 피로도 감소"
- POINT 1·2·3 시리즈에서 안전·인체공학을 포인트 2/3로 정면 배치

**우리(크래프트볼트) 활용 포인트**
- 안전스위치(리뷰 만족 키워드 #7) + 한 손 조작(만족 #6)을 **첫 화면 노출** 필수
- 한국소비자원 49.4% 통계를 인용해 "팔/손 부상 절반 — 안전스위치가 답입니다" 카피 가능
- DEWALT보다 **한국 시니어·여성 사용 사진**으로 톤 차별화

**인용 출처**

### 2. 저가 배터리·A/S 부재 — "1년 만에 버리는 일회용 공구"

**① 발생 사용 상황**
저가 직구·해외 브랜드 제품을 1~2년 사용 후 **배터리 방전·효율 저하**. 굵은 가지 절단 중 멈춤. 배터리만 교체·수리받고 싶어도 **A/S 창구 없음** → 기계 전체 폐기.

**② 가장 심각하게 느끼는 고객 유형**
- **박정훈 페르소나(56세 과수원, 비중 25%)** — 시즌마다 써야 하는 도구라 내구성·A/S가 생사
- 가성비 중시 일반 가정용 소비자 (저가형 진입 후 후회)
- 농촌 거주자 (A/S 망 접근성 떨어짐)

**③ 경쟁사(DEWALT) 대응 — ❌ 다루지 않음 / 오히려 허들**
- **베어툴(Bare Tool) 판매**: 충전기·배터리 별매 → 디월트 20V MAX 시스템 미보유 고객은 추가 구매 비용 부담
- 한국 소비자가 중요시하는 **국내 A/S 메시지·배터리 개별 보증 안내 부재**
- 케이스도 종이박스 — 장기 사용·운반 보호 메시지 약함

**우리(크래프트볼트) 활용 포인트** ⭐ 가장 큰 차별 카드
- **국내 A/S + 마끼다 21V 호환 + BLDC 모터 + 배터리 풀세트** = 4중 안심 메시지
- 박정훈 구매 결정 모멘트 = "BLDC + 국내 A/S + 가격" 3박자 → 첫 화면에 빠지면 이탈
- 카피: "1년 후 버리지 마세요 — 배터리만 갈면 5년 더 씁니다"
- DEWALT가 절대 못 따라오는 영역 (자사 록인 전략이라 호환 카드 못 씀)

**인용 출처**
- A: NotebookLM 답변 §2 ("저가형 배터리 효율 저하 및 A/S 수리 불가")
- B: review_keywords_analysis.md 만족 #9 "BLDC + 국내 A/S" (이*근 인용), 불만 #4 배터리 한계
- C: market_analysis_report.md ②경쟁 비교 — "마끼다 호환 + 국내 A/S = 빅4가 못 메꾸는 틈", 박정훈 페르소나

### 3. 체인 이탈 + 수동 급유 — "5분마다 작업 흐름 끊김"

**① 발생 사용 상황**
- **체인 장력 이탈**: 1시간 정도 연속 작업 시 체인이 늘어나 가이드바에서 이탈 또는 풀림
- **수동 급유**: 자동 급유 시스템 부재로 **약 5분/40컷마다** 작업 멈추고 체인·바에 오일 수동 도포
- 과수원·정원에서 수십 개 가지를 연속 처리할 때 효율 직격

**② 가장 심각하게 느끼는 고객 유형**
- **박정훈 페르소나(과수원)** — 시즌 집중 작업 시 30분 이상 연속 작업
- 전문 조경업자 / 농부 (반복적·연속적 작업이 본업)
- **김지영 페르소나** — 첫 사용 시 "오일이 박스에 안 들어 있어서 별도 구매" → 별점 1·3·4점 직격

**③ 경쟁사(DEWALT) 대응 — ⚠️ 부분적 다룸**
- "**바 조임 노브 & 체인 장력 조정**": 작업 목적별 세밀 조정 + 렌치 기본 제공
- **장점**: 장력 조정 자체는 강조
- **약점**: 무도구(Tool-free) 방식이 아니라 여전히 렌치 필요. 자동 급유 언급 없음

**우리(크래프트볼트) 활용 포인트** ⭐ 즉시 평점 회복 영역
- **체인 오일 충분 동봉** + **출고 전 체인 텐션 사전 조정** + **신품 검수 완료** → 별점 1·3·4점 즉시 차단
- 카피: "오일 박스에 다 들어 있습니다. 출고 전 텐션 다 잡아 두었습니다."
- 자동 급유 시스템이 있으면 **시장 카테고리 선두 차별점**으로 카피화 (DEWALT도 없음). 없으면 "여분 오일 200ml 동봉" 같은 정량 약속

**인용 출처**
- A: NotebookLM 답변 §3 ("작업 중 체인 이탈, 번거로운 수동 급유")
- B: review_keywords_analysis.md 불만 #1 "체인 오일 미동봉", #2 "체인(톱 줄) 늘어남·풀림" (각 2건)
- C: market_analysis_report.md ②경쟁환경 약점 — "체인 오일 미동봉, 체인 늘어남, 출고 검수 미흡"

## 🟡 유력 Pain Point (2소스 일치)

### 4. 굵은 목재(20cm↑) 마력 한계 — "기대치와 다르면 별점 폭락"

**발생 상황**: 8인치(20cm) 미니 체인톱은 직경 15cm 이하가 적정. 20cm 이상은 마력 한계로 천천히 작업해야 하나, 마케팅에서 "벌목용"으로 과장하면 기대치 미스매치.

**가장 심각한 고객 유형**: 박정훈(과수원), 정한수(목공) — 베테랑 사용자가 한계 인지하지만 광고가 과장하면 즉시 별점 폭락

**경쟁사 대응**: DEWALT는 "100×100mm 목재 최대 70컷" 정량 명시 — 기대치 정렬 우수

**우리 활용**: "**과수원·정원·가지치기 전용 (15~20cm 권장)**" 명확화. "벌목" 단어 절대 사용 금지 (research_brief.md ⑤ 명시)
