# 크래프트볼트 21V 무선 미니 전기톱 — 디자이너 6섹션 기획안

> 모바일 세로 스크롤(360–414px) 기준. 한국 시니어·여성 톤. 브랜드 포인트 `#FF6B00`.

## 1. 히어로 — "전선의 끝, 무선의 시작"
- **레이아웃**: full_width 1:1 정사각 + 하단 3컬럼 인포 스트립
- **감정 톤**: Anxiety → Relief (전환의 순간)
- **배경색**: `#F8F9FA` (라이트 그레이, 무채 톤으로 시선 분산 차단)
- **시각 처리**:
  - 상단 **좌우 분할 비교 컷**: 좌측은 채도 30% 다운된 무채색 — 90m 전선이 과수원 바닥에 엉킨 장면(불안). 우측은 풀 컬러, 한국 50대 남성이 **한 손**으로 작업하는 컷(해방).
  - 헤드라인 **48pt Bold** "전선 풀고, 엔진 내려놓고" — 좌측 비교 위에 검정 텍스트, 우측 인물 위에 `#FF6B00` 1단어 강조("무선").
  - 하단 인포 스트립 3개 아이콘(2.0kg / 21V / 8인치) — 24pt 숫자, 12pt 단위. 라인 아이콘만(컬러 노이즈 제거).

## 2. 안전 보증 — "팔·손 부상 절반, 그래서 안전스위치"
- **레이아웃**: split_50_50 (좌 다이어그램 / 우 통계)
- **감정 톤**: Relief (불안의 직접 응답)
- **배경색**: `#E8F5E8` (연한 그린, 의료·안심 코드)
- **시각 처리**:
  - 좌측: 손잡이 **확대 단면도** + 빨간 화살표로 안전스위치 위치 콜아웃. "오작동 차단 1단계" 라벨 14pt.
  - 우측: **49.4%** 숫자를 72pt 검정으로 도넛 차트 중앙에 배치. 도넛 절반은 `#FF6B00`, 나머지는 `#D0E8D0`. 하단 출처 10pt 회색 "한국소비자원 / 기계톱 사고 356건".
  - 하단 풀폭: 한국 30대 여성이 **한 손**으로 가지치기하는 사진 — 자연광 평일 오후 톤(외국인 산악 톤 회피).

## 3. 성능 실증 — "도검형 톱날, 15~20cm가 우리 영역"
- **레이아웃**: 상단 풀블리드 사진 + 하단 4단 게이지 인포그래픽
- **감정 톤**: Confidence (성능 신뢰)
- **배경색**: `#FFF8E1` (따뜻한 옐로, 햇볕·작업감)
- **시각 처리**:
  - 상단: 톱날 끝이 직경 10cm 목재를 **관통하며 빠져나오는 측면 사진**. 톱밥은 동결된 입자처럼 또렷이. 화면 우상단 67세 사용자 얼굴 컷-인 + 큰따옴표 인용 "도검처럼 좁아진" 16pt 손글씨체.
  - 하단 4단 게이지: 5cm·10cm·15cm·20cm 가지 단면 사진을 가로 정렬. 5–15cm는 `#4CAF50` 체크, 20cm는 `#FFC107` 주의 아이콘, 25cm 이상 영역은 회색 처리(어두운 색 X — 부정 회피). "권장 절단 굵기" 라벨이 시각 위계 1순위.

## 4. 배터리/스펙 — "마끼다 21V 그대로, 두 시간 한 호흡"
- **레이아웃**: card_grid 2x2 + 하단 가로 타임라인
- **감정 톤**: Confidence (불안 잔여 제거)
- **배경색**: `#E3F2FD` (차분한 블루, 기술·신뢰 코드)
- **시각 처리**:
  - 카드 4개: ① BLDC 모터 단면 일러스트 ② 마끼다 호환 배터리 클로즈업(브랜드 로고는 블러 처리) ③ 4.0Ah 라벨 매크로 ④ 충전 어댑터. 각 카드 흰색 배경, 그림자 `0 2px 8px rgba(0,0,0,.06)`.
  - 하단 가로 타임라인: 10:30 → 13:00 시간축에 **실제 작업 진행률 막대** `#FF6B00`로 채움. "여분 1개로 반나절" 캡션 14pt.
  - 카드 라벨은 한국어 큰 글씨, 영문 스펙은 보조 10pt — 시니어 가독성 우선.

## 5. 가치 제안 — "박스 열면 끝, 1년 뒤에도 끝나지 않음"
- **레이아웃**: 상단 풀세트 오버헤드 샷 + 하단 체크리스트 카드
- **감정 톤**: Desire (소유 충동)
- **배경색**: `#FFF3E0` (따뜻한 오렌지, 환영·따뜻함)
- **시각 처리**:
  - 상단: **천 위에 일렬 정렬된 풀세트 오버헤드 컷** — 본체·배터리·충전기·체인 오일 250ml·케이스. 각 구성품 위에 작은 흰색 라벨로 이름 표기. 케이스만 살짝 열어 내부 정돈 노출.
  - 하단 체크리스트: `#FF6B00` 체크 아이콘 5개 — "본체 / 배터리 / 충전기 / 오일 250ml / 하드케이스". 그 아래 **국내 A/S 보증 마크 원형 배지**(테두리 2px `#FF6B00`, 안쪽 흰색, "국내 A/S 보증" 14pt).
  - DEWALT 베어툴 비교 미니 표 — 단 2행, 흑백 처리로 시각 위계 낮춤.
