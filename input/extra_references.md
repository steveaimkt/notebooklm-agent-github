**URL**: https://www.kdsn.co.kr/mobile/article.html?no=26627 (한국재난안전뉴스)
**요약**: DIY족 증가로 전기톱 사용 급증, 안전사고 예방 가이드
**인용**: "한국소비자원에 따르면 최근 3년간(2018~2020년) 소비자원에 접수된 기계톱 관련 안전사고는 총 356건으로, 팔·손이 176건(49.4%), 둔부·다리·발이 157건(44.1%), 머리·얼굴이 16건 순"
**활용 포인트**: 안전스위치 강조 카피의 정량 근거. "팔/손 부상 49.4% — 안전스위치가 답입니다" 같은 카피 가능

**인용**: "체인이 둔탁할 경우 재료를 절단하지 못해 리베이트 현상 → 근로자 얼굴·머리 부상 위험"
**활용 포인트**: "안심 섹션"에 인용 가능. KOSHA 출처 = 신뢰도 ★★

## ② 인증·규제

### KC 안전인증 — KTR (한국화학융합시험연구원)
**URL**: https://www.ktr.or.kr/certification/kc/contentsid/433/index.do
**요약**: KC인증 제도 공식 안내 — 전기용품 안전인증 절차/대상
**인용**: "전기용품·생활용품·어린이제품 대상 안전성 인증, 출고 전 또는 통관 전에 모델별 인증 필수"
**활용 포인트**: 상세페이지 "안전 인증" 섹션 — KC 마크 + 인증번호 표기로 신뢰도 ↑

**URL**: https://kats.go.kr/content.do?cmsid=44

**활용 포인트**: 무선 전기톱은 18V/21V 충전식 → 전기용품 안전기준 확인 후 KC 인증 표시 필수

### 안전인증 표시 가이드라인

**활용 포인트**: 상세페이지에 KC 마크 이미지 + 인증번호 정확히 표기 (위반 시 광고법 제재 위험)

**활용 포인트**: DEWALT·마끼다 KC 인증 여부 검증 → 우리만 가진 인증이 있다면 차별 카피화
