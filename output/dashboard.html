④ 핵심 인사이트 3가지

1USP는 성능이 아니라 "케이블·엔진톱 대체재"

만족 TOP 1·5·6이 모두 편의성(무선·경량·한 손). 다수 사용자가 "큰 엔진톱은 자주 쓸 일 없을 것" "큰 전기톱은 위험·무거워서 처박아둠"으로 명시. 메시지 우선순위 = 무선 → 경량 → 한 손 → 그다음 절삭력.

2"15~20cm 가지치기 도구" 사용 맥락 경계선이 명확

긍정 리뷰는 모두 15cm 이하 가지치기·과수원·정원 맥락. 한*국(5★, 67세 목공 베테랑)조차 "20cm 이상은 마력 한계로 천천히". "대형 벌목"으로 메시지 잡으면 별점 폭락 위험. 상세페이지에 적정 굵기 명시 권장.

3별점 1·3·4의 핵심은 "기본기 결함"(오일·체인·검수)

불만 6건 중 4건이 오일 미동봉 + 체인 늘어남 + 사용감 있는 제품 수령. 절삭력·무선 같은 큰 강점은 모두 인정받음. 즉시 개선 가능한 영역만 잡으면 평점 평균 4.0 → 4.5 회복 + 구매전환 직접 상승.

⑦ 기회 3 · 리스크 3

기회 영역

1. 도심 정원 무선톱 포지셔닝 — 한국 저소음 정책(가솔린 100dB → 무선 80dB) + 셀프인테리어 18조 원. 김지영 페르소나 33% 비중 → 마케팅 자원 우선 배치.

2. 빅4 못 메꾸는 "중저가 + 마끼다 호환 + 국내 A/S" — Bosch·DeWalt·Milwaukee는 록인이라 호환 카드 못 씀. 박정훈 페르소나의 결정 요인과 일치.

3. 시니어·과수농 직접 채널 — 농업기술센터·농협·농업 유튜버 협업. 빅4 미진입 영역 선점 가능.

리스크

1. 빅4 미니 체인소 진입 — 밀워키 M18 FUEL이 글로벌 1위. 대응 = 호환·국내 A/S 해자 강화.

2. 기본기 결함이 평점·구매전환 타격 — 별점 1·3·4점 4건 모두 오일·체인·검수. 4.5 미만 구매전환 급락.

3. "대형 벌목 메시지" 기대치 미스매치 — 우리 제품은 15~20cm 적정. 대응 = 적정 굵기 명시.

2광고 카피 분기네이버 농업 카페·유튜브(박정훈) / 인스타·홈가드닝(김지영) / 시니어 커뮤니티(정한수)채널·페르소나 매칭
3제품 개선체인 오일 충분 동봉 + 출고 전 텐션 조정 + 출고 검수 강화평점 4.0 → 4.5 회복
4라인업 확장21V 배터리 플랫폼 공유하는 2~3개 공구 추가 (송풍기·드릴 등)헤비 사용자 LTV 2~3배

⑨ 메시지 가이드 — 강조 vs 금지

반드시 강조할 키워드 5

무선 BLDC 모터 마끼다 21V 호환 8인치 / 한 손 / 가벼움 국내 사후 품질관리

→ 케이블·엔진톱 대체재 가치를 가장 직접 전달하는 5개. 박정훈·김지영·정한수 모두에게 통함.

절대 쓰지 말아야 할 표현 3

"대형 벌목용·전문가용"
→ 우리 제품 한계는 15~20cm. "벌목"은 기대치 미스매치 → 별점 폭락.

"가성비 최고·역대급"
→ 시니어 페르소나는 광고 톤 회의적. 구체 근거(BLDC·A/S·호환)가 효과적.

"무한 작업·올데이 사용"
→ 4A 1개 약 2시간. 과장하면 후속 별점 1·3점 직격. 구체 수치 제시가 신뢰.

⑩ 한 줄 요약 (다음 에이전트 전달용)

"케이블·엔진톱의 무게와 불편에서 벗어나고 싶은, 과수원·정원·목공 사용자"

30~60대, 가성비와 국내 A/S를 따지며, 무선·경량·한 손 조작을 첫 번째로 본다. 3개 페르소나(박정훈 25% / 김지영 33% / 정한수 17%)가 70%+ 커버.
