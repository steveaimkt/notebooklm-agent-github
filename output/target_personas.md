## 페르소나 1. 박정훈 — 과수원 운영 시니어 (소규모 농가)

### 프로필
| 항목 | 내용 |
|---|---|
| 나이/성별 | 56세 / 남 |
| 직업·거주지 | 사과·복숭아 과수원 운영 / 경북 의성 (시골 거주, 차량 보유) |
| 라이프스타일 | 봄·가을 시즌 집중 작업. 가성비 따짐. 농가 커뮤니티 활동 |
| 디지털 활용 | 네이버 카페·밴드, 유튜브 농기계 리뷰. 인스타그램은 거의 안 씀 |

### 구매 동기
- 기존 90m 케이블식 전기톱이 너무 불편 (전선 풀고 이동)
- 엔진톱은 무겁고 위험하며 도심·주거 지역에서 소음 부담
- **케이블/엔진톱의 대체재**가 절실히 필요

### 핵심 니즈 (우선순위)
1. **무선·이동 자유** — 과수원 동선 완전 제약 해방
2. **BLDC 모터** — 내구성·수명 (시즌마다 써야 함)
3. **국내 A/S·배터리 보증** — "사용하다가 버릴 일이 없다"
4. **가격 만족** — 가성비 인식 (엔진톱 대비)

### 가장 큰 불만 포인트
- 배터리 지속시간 (반나절 작업 시 1개로 부족)
- 굵은 가지(20cm↑) 처리 시 마력 부족
- 체인 유격 자주 조절해야 함

### 추천 마케팅 메시지
> "전선과 엔진의 무거움에서 해방되는, 과수원의 무선 일꾼 — BLDC 모터 + 국내 A/S로 시즌마다 든든하게."

### 근거 리뷰
- 이*근(5점): 90m 전선 → 무선 전환, BLDC·A/S·가격 강조
- 3번(4점): 복숭아 과수원, 큰 전기톱 위험성·무게 회피
- 정*석(5점): 반나절 작업 패턴, 여분 배터리 운용

### 6단계 구매 여정

| 단계 | 채널 | 찾는 정보 | 감정 | 리뷰 근거 |
|---|---|---|---|---|
| **① 인지** | 농가 동료 대화, 네이버 농업 카페·밴드, 유튜브 농기계 리뷰 | 기존 케이블·엔진톱의 대안이 있나? | 불편함·피로 누적 | 이*근: "90미터 긴 전선… 너무 불편" |
| **② 탐색** | 네이버 쇼핑 검색("무선 전기톱"·"미니 체인톱"·"과수원 전기톱"), 카페 후기, 유튜브 사용 영상 | 실제 작업 능력, 배터리 시간, 가격대 | 호기심·반신반의 | 3번: "설마 저렇게 두꺼운것도 잘릴까 싶었는데" |
| **③ 비교** | 상세페이지·리뷰, 다나와/쿠팡 가격 비교, 농가 지인 추천 | ① BLDC vs DC ② 배터리 호환성·A/S ③ 톱 길이·마력 ④ 가격 | 신중함 — "버릴 일 없는 도구를 사고 싶다" | 이*근: "BLDC모터… 배터리 A/S, 국내 사후 품질관리… 가격도 마음에 들었구요" |
| **④ 구매** | 네이버 쇼핑/쿠팡, **배터리 2개 옵션** 선택 | 정품 인증, 빠른 배송 | 결단 + 약간의 불안 | 이*근·정*석 "본품 + 배터리 2개이상" 옵션 |
| **⑤ 사용** | 실제 과수원 작업 (시즌 집중) | 1회 충전 시간, 굵은 가지 가능 여부 | 만족·놀라움 + 한계 인지 | 이*근: "한마디로 대박입니다" / 정*석: "1시 넘으니 밧데리 약해" |
| **⑥ 재구매·추천** | 농가 동료 입소문, 같은 21V 플랫폼 다른 공구 | 추가 배터리, 체인 교체 | 신뢰·소속감 | 이*근: "큰 엔진톱이나 일반 전기톱은 자주 사용할 일이 없을것 같다" |

**구매 결정 핵심 모멘트:** ③ 비교 단계에서 **"BLDC + 국내 A/S + 가격"** 3박자. 이 세 정보가 상세페이지 첫 화면에 빠지면 박정훈은 이탈한다.

## 페르소나 2. 김지영 — 도시 정원 관리 DIY 입문자

### 프로필
| 항목 | 내용 |
|---|---|
| 나이/성별 | 38세 / 여 |
| 직업·거주지 | 회사원 (마케팅·사무직) / 경기도 신도시 단독주택 또는 전원주택 입주 2년차 |
| 라이프스타일 | 마당·옥상 정원 셀프 관리. 셀프인테리어·홈가드닝에 관심. 도구는 처음 |
| 디지털 활용 | 유튜브·인스타그램(DIY·홈인테리어 콘텐츠), 네이버 블로그 후기 |

### 구매 동기
- 정원의 작은 나무·잔가지 정리. 손톱은 힘들고 큰 엔진톱은 부담
- "내가 직접 할 수 있는 가벼운 도구"를 원함
- 무선이라 케이블 신경 안 써도 되는 점이 결정적

### 핵심 니즈 (우선순위)
1. **사용 간편함** — 설명서 없이도 바로 쓸 수 있어야
2. **가벼운 무게** — 한 손으로 들 수 있는 부담 없음
3. **안전스위치** — 혹시 모를 사고 방지 (입문자라 더 민감)
4. **무선** — 정원 어디서나 자유롭게

### 가장 큰 불만 포인트
- **부속품 누락(오일 미동봉)** 같은 기본 검수 결함
- 첫 도구 사용자라 "당연히 들어 있을 것" 기대치 높음
- 검수 안 된 제품 받으면 즉시 별점 1·3점

### 추천 마케팅 메시지
> "전기톱이 처음이어도 5분이면 정원이 정돈됩니다 — 가볍고, 안전스위치 있고, 케이블 걱정 없는 무선 미니톱."

### 근거 리뷰
- 최*지(5점): "전기톱을 처음 써봤는데 사용법이 너무 간편해서 바로 적응"
- 최*민(5점): "전기톱을 처음 사용해 봤는데… 무게도 가볍고 / 정원 관리할 때 자주 사용할 것 같습니다"
- 김*민(5점): "무선이라 정원 작업할 때 매우 편리"
- 김*수(4점, 불만): "오일 넣는 공간은 있는데 오일이 없어… 별도 구매를 해야 되네"
- 6번: "8인치라서 힘이 어떨지 궁금했는데" — 첫 사용자 시각

## 페르소나 3. 정한수 — 시니어 목공·취미 마니아 (어워드 리뷰어)

### 프로필
| 항목 | 내용 |
|---|---|
| 나이/성별 | 67세 / 남 |
| 직업·거주지 | 은퇴자 (전직 사무·기술직). 전원생활 즐김 |
| 라이프스타일 | 베이비부머 세대. 시간 여유 많음. 목공예·가구 만들기 취미. 자녀에게 만들어 선물도 함 |
| 디지털 활용 | 네이버 쇼핑·블로그, 유튜브 제품 리뷰. **본인도 긴 리뷰를 작성, 다른 사용자에게 영향력 행사** |

### 구매 동기
- 일반 절단이 아니라 **"관통력"·"톱날 형태"** 같은 기술 디테일에 끌림
- 사진만으로도 다른 제품과의 차별점을 식별
- 깊이 있는 사용 만족감 + 자기 안목 검증 욕구

### 핵심 니즈 (우선순위)
1. **관통력·절삭 성능** — 35cm까지 절단 가능 같은 한계 돌파
2. **안전스위치** — 오작동 방지 (시니어라 더 민감)
3. **한 손 조작** — 정밀 작업
4. **비교 우위가 명확한 차별점** — 사진·스펙으로 식별 가능해야

### 가장 큰 불만 포인트
- 굵은 목재(20cm↑) 절단 시 소형 장비 마력 한계 인지
- 제품 자체보다는 "초보자가 쉽게 사기 어렵다"는 인식 — 양면성

### 추천 마케팅 메시지
> "도검처럼 좁아진 톱날 형태 — 절단을 넘어 관통까지. 35cm 목재도 가능한 8인치 무선 미니톱의 차별점."

### 근거 리뷰
- 한*국(5점, 67세, 도움 11명): 목공예 경력자, 관통력 분석, 35cm 절단 사진 첨부, 톱날 형태 차별점 식별
- 바다동산(5점, 도움 4명): "여기저기 비교해보고 고르고 골라서 / 안전스위치가 있다는거 가장 맘에들었다"

## 다음 단계 시사점 (Step 4 종합 리포트용)

1. **상세페이지 구조**: 박정훈(BLDC·A/S) → 김지영(가벼움·안전) → 정한수(관통력) 순으로 메시지 배치하면 모든 페르소나 커버 가능
2. **광고 카피 분기**: 농업 카페/유튜브 → 박정훈 메시지 / 인스타·홈가드닝 → 김지영 / 시니어 커뮤니티 → 정한수
3. **즉시 개선 영역**: 오일 동봉 + 체인 텐션 사전 조정 + 출고 검수 (3개 페르소나 모두 평점 상승 효과)
